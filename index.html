Learn How to Write KidScript!

KidScript is easy! Here are some simple commands to get started:

  • make: Create a variable. Example: make number 5
  • show: Display something. Example: show number
  • repeat: Repeat an action. Example: repeat 5 times show "*"
  • Combine variables and strings: show "I love " + name
